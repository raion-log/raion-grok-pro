🔒 개인정보처리방침

RAION Grok Pro Chrome Extension

최종 수정일: 2026년 1월 29일

1. 개요

RAION Grok Pro(이하 "확장 프로그램")는 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 확장 프로그램이 어떤 정보를 수집하고, 어떻게 사용하는지 설명합니다.

핵심 요약: 이 확장 프로그램은 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다.

2. 수집하는 정보

본 확장 프로그램은 다음과 같은 정보만을 로컬에 저장합니다:

  • 사용자 설정: 종횡비, 비디오 재생 시간, 업스케일 옵션 등 사용자가 선택한 환경설정
  • 임시 작업 데이터: 비디오 생성 작업 중 필요한 임시 데이터 (작업 완료 후 자동 삭제)

위 정보는 모두 사용자의 브라우저 로컬 스토리지에만 저장되며, 외부로 전송되지 않습니다.

3. 수집하지 않는 정보

본 확장 프로그램은 다음 정보를 수집하지 않습니다:

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 사용자가 입력한 프롬프트 내용
  • 업로드한 이미지 파일
  • 생성된 비디오 파일
  • 브라우징 기록
  • grok.com 계정 정보

4. 데이터 저장 위치

모든 데이터는 사용자의 브라우저 내 chrome.storage.local에만 저장됩니다. 이 데이터는:

  • 사용자의 기기에서만 접근 가능합니다
  • 외부 서버로 전송되지 않습니다
  • 확장 프로그램 삭제 시 함께 삭제됩니다

5. 제3자 공유

본 확장 프로그램은 어떠한 데이터도 제3자와 공유하지 않습니다.

6. 외부 서비스 연결

본 확장 프로그램은 다음 외부 서비스에 연결됩니다:

  • grok.com: 확장 프로그램의 핵심 기능 제공 (xAI에서 운영)
  • status.x.ai: Grok 서버 상태 확인 (선택적 기능)

이러한 연결은 확장 프로그램의 기능 제공에만 사용되며, 사용자 데이터를 전송하지 않습니다.

7. 권한 사용 목적

본 확장 프로그램이 요청하는 권한과 그 사용 목적:

  • sidePanel: 사용자 인터페이스를 사이드 패널로 표시
  • activeTab: 현재 탭에서 사용자가 시작한 작업 실행
  • tabs: 페이지 전환 감지 및 작업 상태 유지
  • storage: 사용자 설정을 로컬에 저장
  • host_permissions (grok.com): grok.com에서만 확장 프로그램 작동

8. 아동 개인정보

본 확장 프로그램은 13세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

9. 개인정보처리방침 변경

본 개인정보처리방침은 법률 또는 서비스 변경에 따라 수정될 수 있습니다. 중요한 변경 사항이 있을 경우, 확장 프로그램 업데이트 또는 이 페이지를 통해 안내드립니다.
